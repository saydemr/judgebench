# Contextual Benchmarking Evaluation Code

Minimal and cleaner code of the paper [Benchmarking contextual understanding for in-car conversational systems](https://arxiv.org/abs/2512.12042) from one of the co-authors.

Original repo can be found [here](https://github.com/JUDGE-Bench/code). Note that that repo contains **a lot of stale code**, which is not needed for reproducing the results in the paper. This repo contains only the necessary and sufficient code to reproduce the results.
